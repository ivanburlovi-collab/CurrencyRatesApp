# 💵 CurrencyRatesApp

Приложение для просмотра курсов валют Центрального Банка России.

## 📋 Описание

Приложение позволяет:
- 💰 Просматривать **текущие курсы** валют ЦБ РФ
- 📅 Получать **архивные курсы** на любую дату
- 🎯 Работает с 10 валютами: USD, EUR, CNY, GBP, BYN, JPY, KZT, UAH, TRY, CHF

## 🛠 Технологии

- **Язык:** C#
- **Интерфейс:** Windows Forms
- **API:** Bank of Russia (официальный)
- **Формат данных:** XML

## 🚀 Установка и запуск

1. Скачать проект
2. Открыть в Visual Studio
3. Нажать F5

## 🌐 Используемый API

- Текущие курсы: `https://www.cbr.ru/scripts/XML_daily.asp`
- Архивные курсы: `https://www.cbr.ru/scripts/XML_daily.asp?date_req=ДД.ММ.ГГГГ`
